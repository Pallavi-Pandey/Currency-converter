# Currency Converter

A simple, real-time currency conversion tool built with static web technologies.

## Project Structure

- `index.html`: The main user interface for selecting currencies and entering amounts.
- `main.js`: Core logic for fetching exchange rates and performing calculations.

## Key Features
- **Real-time Conversion**: Fetches live data from public exchange rate APIs.
- **Interactive UI**: Instant updates as you type or change currency pairs.
- **Lightweight**: Minimalist design with no backend required.

## Getting Started
Open `index.html` in any modern web browser to start converting.
